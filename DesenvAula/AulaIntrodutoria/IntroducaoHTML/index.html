Tags de título

São tags que definem diferentes formatações para o título. Possibilitam que o conteúdo da página seja categorizado de acordo com a relevância do título.

Tags para corpo do texto

Estas tags são usadas para dar forma e estilo* ao texto. São consideradas como ferramentas mais básicas, das quais, eventualmente podem ser alteradas dou sobrescritas por CSS ou JS.

* estilizar é algo mais abrangente do que a colocação usual que á atrelada ao CSS e JS. Apesar de ser mais limitadas tem um papel importante na criação do corpo da página.

Funções das tags aprendidas

  1. Variação tags de títulos

    h1

    h2

    h3

    h4
    h5
    h6
    h1 representa título principal da página e a partir do h2 é atribuído a função de subtítulos, ou títulos de textos secundários.

  2. Tags para textos

    Tag <p></p> são usados para definir um paragrafo de textos.

    "Isso é um paragrafo"

    Tag <mark></mark> Usados para marcar(grifar) uma palavra ou frase importante do texto.

    "Isso é um paragrafo com marcação em amarelo."

    Tag <blockquote></blockquote>

    "Isso é um elemento de bloco com quebra de texto, adiciona a um paragrafo e estende por toda a linha."

    As tags <i> </i>, <b> </b> e <u></u>, produzem os efeitos de itálico, negrito e sublinhado.

    Eventualmente indica-se utilizar a tag negrito(<b> </b>) em casos onde não haja outra tag apropriada.

    Tag <strong> </strong>

    É uma tag parecida com o negrito, e caracteriza um texto importante. Aconselha-se usar a tag <b> </b>, quando especificar um texto sem importancia.

Tags sugeridas para estudo

As tags sugeridas para estudos <font>, <del> e <abbr>, foram pesquisadas no site w3schools.com

  1. Tag <font> </font>: Esta descontinuada a partir do HTML5, fortemente utilizado no HTML4, sua função é semelhante a sintaxe do CSS in-line atuando na estilização do texto sobre as cores família da fonte e tamanho.

    Exemplo: Texto estilizado com tag font

  2. Tag <del> </del>: Esta tag indica uma texto excluído, produzindo um efeito de marcação em tachado. Pode ser utilizado com a tag <ins> </ins>, que marca o texto com um sublinhado

    Exemplo: texto cancelado

    texto substituto.

  3. Tag <abbr> </abbr>: Esta tag é usada para identificar uma abreviação no texto. É uma tag de função semântica, e sua principal utilidade é contribuir na responsividade quando um usuário necessitar de que o navegador leia a tela.
